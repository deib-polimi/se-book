# se-book
This repository includes additional material associated to the book M. Camilli, E. Di Nitto, A. Fuggetta, A. Margara, M.G. Rossi, D.A. Tamburri. Software Engineering: a Structured Journey from Inception to Delivery. BUP 2025.  

For updates and news about the book and for general discussions about software engineering, please refer to the website https://www.sepolimi.com/
